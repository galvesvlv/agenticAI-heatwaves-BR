# Data Requirements

The `storage/` directory must contain the following files:

- `era5_t2m_daily_max.zarr`  
  Zarr dataset containing daily maximum 2-meter air temperature (t2m).

- `WMO_heatwave_conditions_1961-present.nc`  
  NetCDF file containing heatwave condition data based on the WMO definition, covering the period from 1961 to the present.
